# 各种用到的配置文件的模版

> This Project works as a repo for all ide config

## vim配置
需要配置全局的~/.vimrc和当前项目的.ide.vim两个文件，每个项目使用自己单独的配置，全局没有任何额外配置
```
cp ./vim/.vimrc ~/.vimrc
cp ./vim/ide.vim .ide.vim
```

## eslint配置

